# KAZUOPOEMS

KAZUOPOEMSは、荒木一夫の詩のサイトです。サイトはReact Staticを使って作成しています。

サイトの技術的なコードは自由にお使いいただいてかまいませんが、詩作品の引用の際はご一報くださいますようお願い致します。
